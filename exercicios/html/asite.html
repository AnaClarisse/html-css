7- Listas

Listas ordenadas são usadas a tags <ol> para inciar e <li> para adicionar os itens

Utilizamos o <start> para escolher a ordem que começa a lista

Tipos de estilo:

  • A
  • I
  • i

Lista desordenada são usadas a tag <ul> para inciar e <li> para adicionar os itens

Tipos de estilo: circle
square

Pode misturar listas

Listas de dicionario são usadas as tag <dl> para inicia, <dt> para o termo e <dd> para a discrição

8- Links

A tag <a> serve para linkar um site/pagina

Utuliza-se 'target="_blank"' no <a> para que o link abra em uma nova janela e 'target="_self' para abrir na mesma janela

O 'rel=""' serve para indicar como vai ser o estilo de navegação da suas paginas:
external
next
prev
nofollow

Para sair de uma pasta para linkar usa-se '../'

Para dowload de arquivos se utiliza 'download="nome do arquivo" type="tipo do arquivo"'

No site iana.org, tem os tipos de arquivos e o que usar no 'type=""'

9- Imagens dinamicas

use o <picture> para inicia, use o 'img' normal, para adicionar a imagem inicial maior ou menor, depois use o 'soucer:media:type' para adicionar a imagem maior se você começar pela menor e vice versa

10- Audio

use <audio> para iniciar, use 'soucer:src' para adicionar o nome do audio e o tipo.

Também se utiliza 'autoplay controls' logo apos o 'preload=""', se utiliza o 'loop' para criar um loop no audio

tipos de audios:

  • .mp3
  • .ogg
  • .wav

Para encontrar audios que podem ser usados, use a biblioteca de audio do youtube

12- Cores

Paletas tem de 3 a 5 cores

laranja, verde e roxo

cores tercearias

vermelho-arroxeado, azul-esverdeado, azul-arroxeado, vermelho-alaranjado, amarelo-alaranjado e amarelo-esverdeado

Cores complementares

Cores contrarias(Contraste)

Cores análogas

Cores ao lado(Harmonia cimetrica)

Cores análogas e complementares

Cores ao lado e uma cor oposta da principal

Cores análogas relacionadas

2 cores uma ao lado da outra, pula uma cor do lada e escolhe a proxima

Cores intercaladas

pula a cor do lado e escolhe a proxima até ter 3 cores

Cores triadicas

Cores que formam um triangulo equilatero

Cores quadrado

pula 2 cores ao lado e escolhe a proxima até ter 4 cores

Cores tetrádicas

Escolhe duas cores e os seus opostos

Monocromia

Escolhe uma cor e modifica a saturação e o brilho(degrade)
