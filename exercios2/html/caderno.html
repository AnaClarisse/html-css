Anotações de consulta

HTML:

Front-ende: Experiencia do usuario (Trabalhar com o usuario diretamente).
Back ende: Seu codigo com o servidor (Trabalhar com/para o servidor ).

Para imagens dinamicas use o <picture> para inicia, use o 'img' normal, para adicionar a imagem inicial maior ou menor, depois use o 'soucer:media:type' para adicionar a imagem maior se você começar pela menor e vice versa

Para audios use <audio> para iniciar, use 'soucer:src' para adicionar o nome do audio e o tipo.

Dentro do <audio>, usa-se 'preload=""':

  • auto:Só carrega o site se o audio carregar

  • metadata:Carrega algumas informações

  • none:Não carrega nada até que o usuario dê o play

Links:

Utuliza-se 'target="_blank"' no <a> para que o link abra em uma nova janela e 'target="_self' para abrir na mesma janela

O 'rel=""' serve para indicar como vai ser o estilo de navegação da suas paginas:

  • external:Paginas extenas

  • next:Proxima pagina está no link

  • prev:Pagina anterior está no link

  • nofollow:Diz para o mecanismo de busca para não relacionar a sua pagina(para propagandas)
